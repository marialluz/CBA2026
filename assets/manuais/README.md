# Manuais Renault

Os manuais do proprietário dos veículos Renault **não estão incluídos neste repositório** por questões de direitos autorais.

Eles também **não são consumidos pelo código deste projeto** — a base de conhecimento usada pelo agente fica hospedada na plataforma ElevenLabs, e é lá que os PDFs precisam ser enviados.

## Como obter os manuais

Baixe os manuais oficiais diretamente do site da Renault Brasil:

**https://www.renault.com.br/manuais.html**

## O que fazer com os PDFs

Após baixá-los, faça o **upload na base de conhecimento do agente na plataforma ElevenLabs**. O agente usará esses documentos via RAG para responder às consultas técnicas dos usuários.
